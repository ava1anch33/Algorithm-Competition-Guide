本仓库用于记录李煜东著《算法竞赛进阶指南》的学习笔记、例题题解以及配套练习代码。旨在通过系统性的刷题，打牢算法基础，构建完整的知识体系。

---

## 📂 目录结构

建议按照书中的章节编号组织代码，方便检索：

* `/0x00_Basic_Algorithms`: 0x01 位运算、0x02 递归、0x05 排序等。
* `/0x10_Data_Structures`: 0x11 栈、0x13 链表、0x17 二叉堆等。
* `/0x20_Search`: 0x21 树与图的遍历、0x24 迭代加深、0x29 总结。
* `/Templates`: 个人整理的高频算法模板。
* `/Notes`: 核心概念的推导及避坑指南。
